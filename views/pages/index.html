What is TigerMaps?

Glad you asked!

We often find themselves overwhelmed by the abundance of events and activities occurring on campus. It can be quite hard to get a good overview of whatâ€™s going on on campus at any specific moment on any given day and helping that inform our decisions on which activities to attend and how to plan out our days. So our team got together and came up with our solution, TigerMaps.

TigerMaps is a web app that provides a live interactive campus map which integrates seamlessly with multiple dynamic data feeds primarily related to events occurring at Princeton University of interest to students, faculty, and visitors.

Try it Out!

Interactive Map

The main page will have a large dynamic interactive map in the middle, where users can pan different areas of the map and click/hover/touch different buildings. Doing so would result in a pop-up or panel showing information about the building.

Try it Out!

Filterable Options

Above the map, there would be a search bar, where users can use queries to search for different buildings, events, or hosting entities. There would also be filters that would limit the search results.

  • Filter/Search by building
  • Filter/Search by academic events
  • Filter/Search with many other categories
  • Data is continuosly retrieved from sources
Try it Out!

Public and Private Data

The search results would initially be limited to public events, i.e. events accessible to the campus community and campus visitors. After the user has logged in via their netID, more search results would show up.
